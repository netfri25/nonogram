# Nonogram
simple nonogram game writtern in C with [raylib](https://www.raylib.com/)

## Building
#### Dependencies
 - gcc
 - [raylib](https://www.raylib.com/)

#### Process
first, build the build tool (nob)
```shell
gcc nob.c -o nob
```

then run it
```shell
./nob
```

after that, you can find your executable at `./build/main`
